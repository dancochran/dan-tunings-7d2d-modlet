# dan-tunings-7d2d-modlet

Overall game tuneup with bigger stacks, bigger backpack and others.


## install

git archive --format=zip --output=dan-tunings-7d2d-modlet.zip HEAD

copy the zip file into steam/steamapps/common/7 Days To Die/Mods and extract into a folder with the mod name (or your choosing)

start the game
